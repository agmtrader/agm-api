AGM API Documentation

All endpoints except /login require JWT authentication. Include the JWT token in the Authorization header:

Authorization: Bearer your_jwt_token

Authentication

POST /login

Authenticate and receive a JWT token.

Request Body:

{
    "username": "string",
    "password": "string"
}

Response:

{
    "access_token": "string"
}

Drive API Documentation

GET /drive/shared_drive_info

Get information about a shared Google Drive by name.

Request Body:

{
        "drive_name": "string"
    }

Response:

{
        "id": "string",
        "name": "string"
    }
GET /drive/folder_info

Get information about a folder within a parent folder.

Request Body:

{
        "parent_id": "string",
        "folder_name": "string"
    }

Response:

{
        "id": "string",
        "name": "string",
        "parents": ["string"]
    }
POST /drive/reset_folder

Delete all files within a specified folder.

Request Body:

{
        "folder_id": "string"
    }
GET /drive/files_in_folder

List all files within a specified folder.

Request Body:

{
        "parent_id": "string"
    }

Response:

[{
        "id": "string",
        "name": "string",
        "parents": ["string"],
        "mimeType": "string",
        "size": "string",
        "modifiedTime": "string",
        "createdTime": "string"
    }]
POST /drive/create_folder

Create a new folder in a specified parent folder.

Request Body:

{
        "folder_name": "string",
        "parent_folder_id": "string"
    }

Response:

{
        "id": "string",
        "name": "string",
        "parents": ["string"],
        "mimeType": "application/vnd.google-apps.folder"
    }
GET /drive/file_info

Get information about a specific file by name and parent folder.

Request Body:

{
        "parent_id": "string",
        "file_name": "string"
    }

Response:

{
        "id": "string",
        "name": "string",
        "parents": ["string"]
    }
PATCH /drive/rename_file

Rename an existing file.

Request Body:

{
        "file_id": "string",
        "new_name": "string"
    }
PATCH /drive/move_file

Move a file to a new parent folder.

Request Body:

{
        "file": {
            "id": "string",
            "parents": ["string"]
        },
        "new_parent_id": "string"
    }
POST /drive/upload_file

Upload a new file to a specified folder. Supports base64 encoded files, CSV data, and more.

Request Body:

{
        "file_name": "string",
        "mime_type": "string",
        "file": "string | array",  // base64 encoded string or array for CSV
        "parent_folder_id": "string"
    }

Response:

{
        "id": "string",
        "name": "string",
        "parents": ["string"],
        "mimeType": "string",
        "size": "string",
        "modifiedTime": "string",
        "createdTime": "string"
    }
DELETE /drive/delete_file

Delete a file from Google Drive.

Request Body:

{
        "file_id": "string"
    }
GET /drive/download_file

Download a file from Google Drive. Optionally parse CSV or Excel files into JSON.

Request Body:

{
        "file_id": "string",
        "parse": "boolean"  // Optional, defaults to false
    }
GET /drive/export_file

Export a Google Workspace file (Docs, Sheets, etc.) to a specified format.

Request Body:

{
            "file_id": "string",
            "mime_type": "string",
            "parse": "boolean"  // Optional, defaults to false
        }
