API Documentation

Authentication

POST /login

Authenticate and receive a JWT token.

Request Body:

{
    "username": "string",
    "password": "string"
}

Response:

{
    "access_token": "string"
}

Reporting Endpoints

POST /reporting/*

Various reporting endpoints for generating and managing reports.

Trade Tickets

POST /trade_tickets/*

Endpoints for managing trade tickets.

Drive Integration

POST /drive/*

Google Drive integration endpoints.

Database Operations

POST /database/*

Database operation endpoints.

Flex Query

POST /flex_query/*

Flexible query endpoints.

Email

POST /email/*

Email-related endpoints.

All endpoints except /login require JWT authentication. Include the JWT token in the Authorization header:

Authorization: Bearer your_jwt_token
